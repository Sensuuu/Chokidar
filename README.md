# Chokidar - Downloads Organizer

Automatically organize downloads into categorized folders using Node.js.

## Features

- Real-time file monitoring
- Extension-based routing
- Nested folder organization
- Collision handling with auto-rename
- Comprehensive logging
- Silent background operation
- Windows auto-start integration

## Install

npm install

## Run

node index.js
